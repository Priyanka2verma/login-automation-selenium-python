# Selenium PyTest POM Automation Framework

## Project Overview
This project is a Selenium automation testing framework developed using Python, Selenium WebDriver, PyTest, and Page Object Model (POM).  
It is created to demonstrate automation testing skills with a clean and structured framework suitable for freshers.

The project automates the login functionality of a demo web application.

## Tech Stack
- Python
- Selenium WebDriver
- PyTest
- Page Object Model (POM)
- WebDriver Manager
- PyCharm

## Application Under Test
URL: https://the-internet.herokuapp.com/login

## How to Run the Project
1. Clone the repository
git clone https://github.com/Priyanka2verma/login-automation-selenium-python.git

2. Navigate to project directory
cd selenium-pytest-pom-automation

3. Install dependencies
pip install -r requirements.txt

4. Run test cases
pytest
